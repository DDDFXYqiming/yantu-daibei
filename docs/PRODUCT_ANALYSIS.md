# 产品分析：为什么做「研途带背」

## 目标用户

用户画像来自聊天记录中暴露出的需求：

- 考研/复试/调剂节点高度焦虑；
- 对学校、专业、分数线、复试权重、调剂成本很敏感；
- 需要把大段讲义和真题转成「能背、能测、能复盘」的东西；
- 在职/兼职状态下时间碎片化，需要明确每日任务；
- 会用打卡小程序，但缺少「资料-计划-带背-复盘」一体化闭环；
- 对低价小额服务有接受可能，如“10块20”的资料定制思路。

## 核心需求

1. **计划拆解**：把“6月底前完成一轮/二轮”“每天30页”“55分钟+5分钟”等文字计划落到每日任务。
2. **资料带背**：粘贴讲义/大纲/真题后自动拆成卡片，以“简述/试述/描述”的形式提问。
3. **复盘闭环**：做题后不能只是记录对错，要让用户标记不会/模糊/会，并按间隔复习。
4. **院校决策**：记录目标院校、分数线、复试比例、成本、证书、调剂风险。
5. **周报输出**：模拟“表格每天记录、每周日晚上给表格”的监督场景。
6. **低价 Pro**：免费体验核心流程，Pro 解锁无限资料、周报和高级决策字段。

## MVP 功能

- 今日任务；
- 55/5 专注；
- 资料粘贴；
- 生成带背卡；
- 到期复盘；
- 院校/调剂决策表；
- Pro 购买占位；
- 本地存储。

## 不做什么

- 不做大而全考研社区；
- 不做盗版资料分发；
- 不承诺录取概率；
- 不做复杂 AI 后端；
- 不做账号系统；
- 不采集敏感个人数据。
